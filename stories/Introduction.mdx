  # @mtes-mct/monitor-ui

  Welcome to the Monitor UI Storybook documentation!

  The `@mtes-mct/monitor-ui` package is shared across 3 applications:

  - [MonitorEnv](https://github.com/MTES-MCT/monitorenv)
  - [MonitorFish](https://github.com/MTES-MCT/monitorfish)
  - [RapportNav](https://github.com/MTES-MCT/rapportnav2)

  This library provides:

  - A global `styled-components` theme.
  - Various common React components.
  - Several common React hooks.
  - A collection of common libraries and utility functions.
  - Various common TypeScript definitions and utilities, as well as some shared Backend entities represented
    by TypeScript interfaces.

  Many of the UI components provided here are based on [React Suite](https://rsuitejs.com) components.

  The designs for most components are available at
  [this Adobe XD reference](https://xd.adobe.com/view/b6d4c472-3fbe-4dec-9f14-38fe03872a3e-e387/grid/).
