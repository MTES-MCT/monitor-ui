# Monitor UI

Welcome to Monitor UI documentation! `@mtes-mct/monitor-ui` is a package shared by 3 applications:

- [MonitorEnv](https://github.com/MTES-MCT/monitorenv)
- [MonitorFish](https://github.com/MTES-MCT/monitorfish)
- [RapportNav](https://github.com/MTES-MCT/rapportnav2)

This library provides:

- A global.
- Common React components.
- Common React hooks.
- Common libraries and utility functions.
- Common Typescript definitions, utilities as well as some shared backend entities.

Many UI components provided here are using [React Suite](https://rsuitejs.com) as base components.

Most components design is available in
[this Adobe XD reference](https://xd.adobe.com/view/b6d4c472-3fbe-4dec-9f14-38fe03872a3e-e387/grid/).
